### 2. **enhancement.py** (Core Processing)
**Purpose**: Image enhancement operations using OpenCV 
**Size**: ~500 lines of code 
**Functions Implemented** (17 total):

| # | Function | Purpose |
|---|----------|---------|
| 1 | `adjust_brightness()` | Brightness adjustment (-100 to 100) |
| 2 | `adjust_contrast()` | Contrast adjustment (0.5 to 3.0) |
| 3 | `apply_gaussian_blur()` | Gaussian blur (1 to 25 kernel) |
| 4 | `apply_sharpen()` | Unsharp masking sharpening |
| 5 | `histogram_equalization()` | CLAHE histogram equalization |
| 6 | `noise_reduction_bilateral()` | Bilateral denoising |
| 7 | `noise_reduction_median()` | Median denoising |
| 8 | `edge_enhancement()` | Laplacian edge enhancement |
| 9 | `auto_enhance()` | One-click automatic enhancement |
| 10 | `color_correction_hsv()` | HSV color space adjustments |
| 11 | `night_mode_enhancement()` | Low-light image enhancement |
| 12 | `flip_image()` | Image flipping (H/V/Both) |
| 13 | `rotate_image()` | Image rotation (any angle) |
| 14 | `get_image_statistics()` | Image statistics calculation |

**Key Algorithms**:
- CLAHE (Contrast Limited Adaptive Histogram Equalization)
- Bilateral filtering
- Gaussian blur
- Unsharp masking
- Laplacian edge detection
- HSV color space manipulation

### 11. **run.sh** (macOS/Linux Automation)
**Purpose**: Automatic setup and launch script

2. Creates virtual environment
3. Activates virtual environment

./run.sh
```

### User Workflow
```
1. Launch app → streamlit run app.py
2. Upload image → File uploader in sidebar
3. Adjust sliders → Real-time preview updates
4. Select filter → From dropdown menu
5. Compare results → Side-by-side/Overlay/Slider

## Getting Started

### Quick Start
```bash
# 1. Navigate to project
cd image-enhancement-tool

# 2. Run setup (automatic)
./run.sh # macOS/Linux
run.bat # Windows

# 3. Open browser
# http://localhost:8501

# 4. Upload and process images!
```

### Manual Start
```bash
# 1. Create virtual environment
python -m venv venv
source venv/bin/activate # or: venv\Scripts\activate (Windows)

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate sample image
python create_sample.py

# 4. Run application
streamlit run app.py
```
